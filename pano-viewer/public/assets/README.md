# Image Assets

## Required Images for Panoviewer Component

1. **Test_Pano.jpg** - Main panoramic image (360-degree photo)
   - Should be an equirectangular projection image
   - Recommended resolution: 4096x2048 or higher
   - Place in `/public/` folder

2. **pin-blue.png** - Marker pin icon
   - 32x32 pixels
   - Blue pin icon for marking locations
   - Place in `/public/` folder

3. **drone.png** - Image layer overlay
   - 220x220 pixels  
   - Drone or custom overlay image
   - Place in `/public/` folder
